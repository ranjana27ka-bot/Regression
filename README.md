# Regression
Using Linear Regression Model predicting the house prices
Python Pandas,Numpy,Matplolib,sklearn modules used in this project
Conclusion
  Preprocessing Steps Performed
  Loaded the California Housing dataset.
  Converted it into a Pandas DataFrame.
  Checked for missing values.
  Split the dataset into training and testing sets.
  Applied feature scaling using StandardScaler.
   
Why These Steps Were Necessary
  Missing value checking ensures data quality.
  Train-test splitting enables unbiased evaluation.
  Feature scaling improves model efficiency and accuracy.

Model Used
Linear Regression

Evaluation Metrics Used
  Mean Squared Error (MSE)
  Mean Absolute Error (MAE)
  R-Squared Score (R²)

Final Observation

Linear Regression provides a simple and effective baseline model for predicting California housing prices. The model's performance can be assessed using MSE, MAE, and R² scores obtained after execution.
